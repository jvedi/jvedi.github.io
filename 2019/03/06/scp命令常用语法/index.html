scp命令常用语法

scp命令

基本格式：
scp file user@host:dir

复制文件到指定端口
scp -P port file user@host:/dir

从远端主机将文件复制到另一台远端主机
scp user1@host:file1 user2@host2:file

从远端主机将文件复制到本地
scp user@host:/远程文件路径 /本地文件存储路径
演示：
scp root@123.123.123.123:/home/storage/jdk.zip /home/local/jdk.zip
